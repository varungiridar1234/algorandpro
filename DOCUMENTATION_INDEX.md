## 💡 Key Concepts

### Risk Score (0-100)
Weighted combination of all environmental factors. Lower = safer conditions.

### Confidence (0-100%)
How reliable is the prediction based on data completeness, variability, and alignment.

### Predicted Yield
Expected harvest output adjusted for all environmental factors.

### Climate Simulation
Scenario testing to show how risk changes under different weather patterns.

### Explainable AI
Human-readable explanations for why the system made its prediction.

## 📞 Troubleshooting

| Issue | Solution |
|-------|----------|
| "Missing required fields" | Ensure all 4 form fields filled |
| Build fails | Check Node version (need 14+) and disk space |
| Blockchain fails | Verify internet connection and testnet availability |
| Slow performance | Clear `npm cache` and `.next/` folder |
| Port 3000 in use | Run `npm run dev -- -p 3001` |

See [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md) for more troubleshooting.
